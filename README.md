# Alinea Immobilien GmbH Website

Produktionsreifes Next.js 14 Projekt mit Tailwind CSS für die Alinea Immobilien GmbH (Frankfurt am Main). Enthält Startseite, Leistungen, Über uns, Kontakt inkl. Formular und Google Maps, sowie rechtliche Seiten (Impressum, Datenschutz, AGB) und eine angepasste 404-Seite. Remote-Bilder werden über HTTPS eingebunden.

## Entwicklung

1. Abhängigkeiten installieren: `npm install`
2. Entwicklung starten: `npm run dev`
3. Produktion bauen: `npm run build`

Tailwind ist konfiguriert (dark mode via `.dark` Klasse). Layout und Komponenten liegen im `app/`- und `components/`-Verzeichnis.
